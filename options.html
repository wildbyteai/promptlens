API 自动分析配置

PromptLens 不内置模型服务。请填写你自己的 OpenAI-compatible Vision API。API Key 只保存在本地浏览器，图片只会发送到你配置的 Base URL。PromptLens 会使用这些 API 字段自动调用你配置的模型服务。

隐私边界： PromptLens 没有自有后端。分析时，请求会发送到你配置的 Base URL，可能包含 API Key、图片 URL 或截图数据、模板提示词、模型名称和业务背景。
右键 vs 框选： 右键网页图片最快，但部分远程图片需要可选站点权限；框选截图更通用，适合防盗链、canvas、blob、登录页或无法右键的内容。
